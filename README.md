# ClownWatch - Clown Report System prototype
Team I's Figma prototype for Software Engineering module project.

The Clown Reporting System or ‘ClownWatch’ is a platform that allows users to report and check clown sightings in their area. 
The system lets users login, sign up or continue as guest, and  provides three functions: report clown sightings, check for clowns in your area, and get information on the last reported clown
